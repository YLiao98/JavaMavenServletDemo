# Basic Java Maven Servlet Demo

Basic demo of a simple servlet application **MavenWebProject**

  - Required packages
    - Tomcat 10.0.18
    - JDK11
    - JAVA IDE (I'm using Intellij 2022 Community Edition)
  - Project structure
    ```
    MavenWebProject/
        - /src
            - /example
                - /yiwei
                    - /test
                        - /LifeCycleServlet.java
        - webapp
            - WEB-INF
                - web.xml
            - index.html
    ```
